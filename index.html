Sobre mim

Oi. Me chamo Thiago. Minha formação foi em Análise e Desenvolvimento de Sistema no IFPE. Apesar de ter tido apenas uma disciplina sobre desenvolvimento front-end foi a área que mais me interessei no curso. Nos meus ultimos 3 anos venho me dedicando a um projeto chamado Libre Designers. Esse projeto visa difundir a aplicação dos softwares livre no design e a aplicação do design na melhoria de softwares livres. Além disso, reservo em média de 1 hora por dia para estudar desenvolvimento de jogos. Venho focando no estudo de Krita, Blender e Godot, mas ainda me encontro em um nível iniciante.

Nas seções abaixo você irá encontrar os meus trabalhos nas áreas de desenvolvimento web e design gráfico tanto realizados em cursos quanto realizados para clientes.

Trabalhos Realizados

Os trabalhos abaixos foram os mais importantes e estão sendo exibidos em ordem cronológica em uma janela de 2 anos por data de finalização. Títulos marcados com * foram realizados para clientes. Clique no título para expandir.

Complex App - 15/02/2021

Complex App envolveu o desenvolvimento front-end utilizando o framework React. O sistema consiste de um esboço de rede social envolvendo cadastro de usuários, possibilidade de seguir usuários para ver suas postagens, busca por postagens, chat usando socket.io além de notificação de recebimento de mensagens de outros usuários .

* Cidauto Store - 02/12/2020

O projeto de Gerenciamento de Veículos CidautoStore envolveu a construção completa do sistema: banco de dados usando migrations, back-end usando laravel e front-end envolvendo LTE Admin. Criou-se uma API para ser consumida pelo site CidautoStore. O sistema envolve 3 níveis de permissão com listagem, cadastro, deleção e edição de quase todos os models.

* Loja Coisas da Darlene - 02/02/2020

Coisas da Darlene é uma loja virtual voltada para o público feminino. A ideia era criar uma loja pequena para disponibilizar objetos a venda da Darlene. Esse projeto passou pela definição do logotipo, design das interfaces da loja e, finalmente, terminando com a integração com o plugin desenvolvido por mim de loja virtual para October CMS.

* Módulos Canal Rural - 15/09/2019

Canal Rural é um blog de notícias do Agronegócio Brasileiro. Minha participação neste projeto foi construir módulos HTML/CSS/JS para tornar mais rápida a construção do projeto como um todo. Criei módulos de widget de previsão de tempo consumindo uma api fornecida, seções da página principal, blocos com gráficos, tabelas informativas de cotação e mais.

Libre Designers 3 - 18/08/2019

Eu vejo o site do Libre Designers como um organismo em constante evolução. Ao contrário dos outros projetos que tive oportunidade de participar, passo 1 ano ou mais para lançar uma nova versão. O Libre Designers é pensado como um blog colaborativo e é desenvolvido em October CMS. Foram criados plugins de redes sociais dos autores e alterados plugins de comentários e status das postagens. No LD3 foi inserido uma nova seção de Novidades no página inicial, opção de visualização de tela clara e noturna, feed.rss para os posts do blog e mais.

Landing Page Pitagoric - 20/04/2019

A Pitagoric é minha empresa individual. Fiz alguns melhoramentos no design do site. Ficou com um visual mais limpo e organizado.
